# Brief Jinu with a Specific Task

When this skill is invoked:
1. Read the user's specific instruction
2. Invoke Jinu
3. Jinu runs `date` and reads context/session-context.md
4. Jinu confirms understanding in one sentence
5. Jinu runs only the relevant agent(s) for the specific task
6. Jinu reports findings and documents to Notion
