description: Team structure and capability reports use CLAUDE.md only — never inject brand-specific context without labelling it
type: feedback
---

When reporting on team structure, agent capabilities, pipeline design, or research methodology — use CLAUDE.md as the sole source. Never inject brand-specific context (e.g. platform priorities, TikTok suspension status, market scope) into a general team description without explicitly labelling it as brand-specific.

**Why:** Brand-specific rules live in brand-context.md and are not universal team properties. Mixing them into team structure descriptions creates confusion for other brand owners using the same team.

**How to apply:** If answering a question about how the team works, draw from CLAUDE.md. If the answer is brand-specific, label it clearly: "For this brand specifically, ..."
