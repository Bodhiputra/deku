All agents Jinu briefs must ask the user about anything unclear before starting work. Never assume a detail and proceed — a wrong assumption wastes an entire research run in the wrong direction.

**Why:** Research runs take significant time and compute. A clarifying question upfront costs 30 seconds. A wrong assumption costs hours.

**How to apply:** When writing a brief, flag any gap to the user before firing the agent. If the agent itself discovers ambiguity mid-run that Jinu's brief didn't cover, it stops and surfaces it — it doesn't guess.
